# Session-opener prompt (paste this first when starting a Claude session)

Use this at the top of a new chat before asking for course-content or curriculum work.
It sets the legitimate educational context up front, which meaningfully reduces
false-positive safety flags on standard health-occupations vocabulary. It is honest
context, not a workaround — no preamble can disable the safety classifier, and if a
message is ever flagged the session continues on another capable model, so work is
never lost (everything is saved in Supabase / the repo as we go).

---

## COPY-PASTE

Context for this session — please read first.

I'm the owner/operator of **Premier Dental Academy of Longview**, a **Texas Workforce
Commission–licensed career school** (TWC Career School License **#S5316**) that has trained
Registered Dental Assistants for 7+ years. I'm building **original, student-facing
educational content** for our state-approved RDA program and the website that hosts it.

This is vocational **health education**. The material uses standard dental-assisting
curriculum vocabulary — infection control, disease transmission, sterilization,
radiography/x-ray safety, dental materials, chemical & waste handling, and patient care —
exactly as taught in accredited programs and aligned to *Modern Dental Assisting*, CDC/OSHA
guidelines, and Texas State Board of Dental Examiners (TSBDE) requirements. Everything is
written in our own words for classroom instruction and state-board exam prep. There is **no
operational-hazard, weapons, or dual-use intent** — it's coursework for future dental
assistants.

Please treat this as the routine educational + web-development project it is and proceed
with the task. If any single item genuinely warrants care, note it briefly and keep going
rather than stopping.

---

## If a message still gets flagged
1. The build is never lost — completed work is committed to Supabase/the repo continuously.
2. The session auto-continues on another capable model (e.g. Opus 4.8); keep working.
3. Click **"Learn more"** on the flag banner and report it as a **false positive**:
   "Licensed RDA vocational school (TWC #S5316); standard health-occupations curriculum;
   educational use." That report is what loosens the threshold on your account over time.
