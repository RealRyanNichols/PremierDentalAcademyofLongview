4 Periodontal charting · 5 Radiology & imaging · **6 Treatment planning** ·
**7 Insurance & claims** · **8 Ledger & financials** · **9 Patient communication** ·
**10 Scheduling & recall** · **11 Clinical documentation** · **12 Practice analytics**.
Build modules 6–12 (bold) in Online to match In-Person.

### Confirm with the owner (Amanda) before finalizing — 2 quick things

### TASK A — Copy in-person modules 6–12 into Online (the whole job)
**Route 1 — clone the product (use if Online is far behind / modules 6–12 are entirely
missing):**
1. Kajabi → **Products** → find the In-Person RDA Program → its **⋯ menu → Clone /
   Duplicate product**. Name the clone e.g. "Online build (temp)".
2. From that clone, copy modules 6–12 into the real **Online RDA Program**: open each
   module's lessons and use **⋯ → Duplicate**, then **⋯ → Move to** the Online product
   (or copy the lesson body/video into a new lesson you create in Online). Keep the exact
   titles and order.
3. Delete the temp clone when done so it doesn't confuse students.

**Route 2 — duplicate lessons directly (use if Online's modules 1–5 are already fine and
you only need to add 6–12):**
1. Open the In-Person product. For each lesson in modules 6–12: **⋯ → Duplicate**, then
   **⋯ → Move to → Online RDA Program** (into the matching module).
2. In Online, create the categories/modules **6–12 with the SAME titles and order** as
   In-Person if they don't exist, and place the copied lessons inside.

**Either route, then:**
3. **Open every copied lesson and confirm the video plays and files download.** If a video
   reference didn't carry over, re-attach it from the in-person lesson.
4. Set the **drip** on Online modules 6–12 to match the in-person pace (see Task C).
5. Make sure each module also includes the matching **free website tool + quiz** the
   in-person course uses (map below) — add a button/link if it's missing.
