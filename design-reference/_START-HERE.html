How this ships to the live site

All 13 pages share one stylesheet (assets/pda.css) and one script (assets/pda.js — the nav, footer, payment calculator, and salary calculator). Change the look once, it updates everywhere. There's a Claude Code prompt in your Premier Dental Academy folder (claude-code-prompt-IMPLEMENT-redesign.md) that tells Claude Code to merge this design system into your real repo, keep your working tools and Square checkout, and open a PR so it deploys to Vercel.
